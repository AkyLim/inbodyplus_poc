# Requirement Specification: Figma to Frontend PoC

## 1. Goal
Implement a frontend feature based on a Figma design using Figma MCP.

## 2. Scope
- Extract design data (tokens, components, layout) from Figma using MCP.
- Implement the design in `apps/frontend` (Next.js).
- Ensure responsiveness and high-fidelity implementation.

## 3. Workflow
1.  **Extract**: Get tokens/assets from Figma.
2.  **Design**: Create local design system / component specs.
3.  **Build**: Implement in Next.js.
4.  **Verify**: Visual comparison.
